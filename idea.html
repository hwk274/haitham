الفكرة

الموقع تم تصميمه ليكون احترافيًا وديناميكيًا، يجمع بين السلايدر للصور والكروت التفاعلية لتقديم خدماتنا بطريقة جذابة وسهلة الاستخدام.

التنفيذ

تم تنفيذ الموقع باستخدام HTML، CSS و JavaScript لتوفير تجربة مستخدم سلسة مع سلايدر ديناميكي وصور متحركة، وأقسام تفاعلية تسهل عرض المحتوى بطريقة جذابة.
